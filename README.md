### Features

- **Bento grid layout** - 12-column responsive CSS grid
- **Cursor glow** - smooth 'requestAnimationFrame' lerp that follows your mouse
- **Comet border trail** - SVG 'animateMotion' 6-dot trail on card hover
- **Typing hero text** - cycles through Full Stack Developer/ Data Scientist / AMP Specialist / SEO Engineer
- **Jakarta Live Clock** - real-time WIB(UTC+7) clock
- **Dark-only theme** — deep purple/indigo, no light mode distraction
